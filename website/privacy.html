Privacy Policy

Last updated: 20 July 2025

1. Scope of Data

MyPrivateRecorder captures only the mixed audio stream of the Zoom meeting that the user explicitly joins via the application. No video, chat messages, participant lists, profile pictures, analytics, or other personal data are collected.

2. Data Storage & Retention

  • The recorded audio is written as a local .wav file to a folder chosen by the user.
  • OAuth / SDK tokens reside in volatile memory and are never logged or written to disk.
  • The application does not transmit data to any external server or third party.

3. Data Sharing

No data is shared, sold, or transferred to third parties. The developer has no access to users’ recordings or tokens.

4. User Control & Deletion

Because all recordings are stored locally, users can review or delete them at any time using standard file‑system tools. Uninstalling the app fully removes the software; no residual data remain.

5. Children’s Data

The app is not intended for use by children under 16 and does not knowingly collect personal data from minors.

6. Contact

Questions or privacy requests can be sent to privacy@myprivaterecorder.local.
